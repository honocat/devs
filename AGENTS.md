# devs workspace

This repository contains personal development projects.

## Directories

apps/
  Application development.
  Includes React Native apps and CLI tools.

article/
  Technical blog articles written in Markdown.

progs/
  Programming practice and experiments.

stats/
  Statistics learning materials using R.

---

## Goals of this repository

1. Build personal software projects
2. Write technical articles
3. Practice programming
4. Learn statistics

---

## Coding guidelines

General:

- Prefer simple and readable code
- Avoid unnecessary dependencies
- Follow existing project structure

JavaScript / TypeScript:

- Prefer TypeScript
- Use functional components
- Avoid class components in React

---

## Article writing rules

Articles are written in Markdown.

When generating articles:

- Use clear explanations
- Include minimal runnable code
- Prefer concise sections
- Target technical readers

---

## When editing code

Before making changes:

1. read related files
2. understand project structure
3. propose minimal changes

Avoid large refactors unless requested.
